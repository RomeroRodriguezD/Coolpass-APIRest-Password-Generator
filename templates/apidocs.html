Coolwords API documentation

Welcome to the Coolwords API documentation. Here you will find information on how to use the API to generate custom passwords.

The endpoint for generating custom passwords is /generate-custom/<int:length>. The length parameter indicates the length of the password to be generated.

In addition to the length parameter, you can also include the following optional parameters to include specific elements in the password:

  • animals - includes animal names in the password
  • colors - includes color names in the password
  • cities - includes city names in the password
  • famous - includes famous people's names in the password

For example, the following URL will generate a password of length 15 with animal names included: /generate-custom/15?animals=true?famous=true.

While this one will generate a password of length 10 with either animals and/or famous people names included: /generate-custom/10?animals=true&famous=true.
